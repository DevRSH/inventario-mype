# Inventario MyPE

API de control de inventario orientada a micro y pequeñas empresas.

## 🚀 Tecnologías

- Python
- FastAPI
- SQLAlchemy
- SQLite

## 📦 Funcionalidades

- CRUD de productos
- Registro de movimientos (entrada/salida)
- Control automático de stock

## ▶️ Ejecución

```bash
pip install -r requirements.txt
uvicorn main:app --reload
